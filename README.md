# Netflix Content Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Netflix content and provides insights into content type, ratings, release years, genres, and countries.

## Tools Used
- Power BI
- Excel / CSV
- GitHub

## Dashboard Features
- Total Content KPI
- Movies vs TV Shows
- Content by Country
- Rating Distribution
- Release Year Trends
- Genre Analysis
